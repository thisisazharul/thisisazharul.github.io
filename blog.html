Questions from Assignment-05

1. What is DOM and what is the purpose of DOM?

The Document Object Model (DOM) is a programming interface for web documents. It represents the page so that programs can change the document structure, style, and content. The DOM represents the document as nodes and objects; that way, programming languages can interact with the page.

2. How will you select HTML elements using DOM? Name the DOM methods.

I will select HTML elements by using any of the six methods provided by JavaScript. The six methods are:

  • getElementById()
  • getElementsByTagName()
  • getElementsByClassName()
  • querySelector()
  • querySelectorAll()
  • name.item()

3. What is event bubble?

Event bubbling is a mechanism in which an event is triggered on a child element and then propagated to its parent element. This process continues until the event reaches the root element of the DOM tree.

Event bubbling is the opposite of event capturing. In event capturing, the event is first captured by the outermost element and propagated to the inner elements. Event bubbling is the default behavior of event propagation in HTML. It can be prevented by using the stopPropagation() method.

4. What is a callback function and why will you use it?

A callback function is a function passed into another function as an argument, which is then invoked inside the outer function to complete some kind of routine or action.

Callbacks make sure that a function is not going to run before a task is completed but will run right after the task has completed. It helps us develop asynchronous JavaScript code and keeps us safe from problems and errors.

Callbacks are used to make sure that certain code does not execute until other code has already finished execution.
